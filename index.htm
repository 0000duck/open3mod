open3mod - Open 3D Model Viewer

View and export all your 3D models. A free, Open-Source tool designed to make it all easier.

Massive 3D Format support

Based on and tightly integrated with Open Asset Import Library, open3mod is able to read more formats than most other viewers. Supported formats include OBJ, 3DS, BLEND, STL, FBX, DXF, LWO, LWS, MD5, MD3, MD2, NDO, X, IFC and Collada, the total number of formats is 40+.

Flexible User Interface

A tabbed User Experience allows opening multiple scenes at once. 3D views can be split in up to four independent viewports. Different camera modi - orbit, first-person-shooter or axis-aligned - are available for each viewport. New scenes are loaded in the background.




Skeletal Animation / Skinning support

Playback skeletal animations in your 3D files. Lock individual frames, adjust playback speed and inspect computed transformations.




Export

Export single textures and whole scenes to a multitude of formats. Mesh export formats include OBJ, PLY, 3DS, Collada and STL. Texture export formats include PNG, JPEG, DDS, TGA, BMP

Hierarchy Inspection

Look at the scenegraph hierarchy in great detail. This is useful for developers who would like to get a better picture of how scenes are organized.
